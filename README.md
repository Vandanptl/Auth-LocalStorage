# ASP.NET 9 + React Authentication with Material UI

A modern authentication system with email/password and social logins (Google, Facebook, Apple) using ASP.NET 9 for the backend and React with Material UI for the frontend.

## Features

- User registration and login with email/password
- Social authentication with:
  - Google
  - Facebook
  - Apple
- JWT-based authentication
- Protected routes
- Responsive Material UI design
- Form validation
- Password visibility toggle

## Technologies Used

### Backend
- ASP.NET 9
- Entity Framework Core
- Identity Framework
- JWT Authentication
- Social Authentication Providers

### Frontend
- React 18
- TypeScript
- Material UI
- React Router
- Axios
- Google OAuth2 Client

## Prerequisites

- .NET 9 SDK
- Node.js (v16 or later)
- npm or yarn
- SQL Server (or other compatible database)
- Developer accounts for social auth providers
